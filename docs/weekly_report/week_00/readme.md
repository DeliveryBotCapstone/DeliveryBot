># 2022-02-15(화) 정기 회의 내용
*매주 목요일 오후 6시 회의 계획*

### 1차 목표
* 기사님 어플(주소 입력) -> TurtleBot3 -> 목적지 문 앞 -> 사용자 어플 알림 -> 택배 놓기 -> 원래 장소로 돌아오기

### 추가
1) 엘리베이터 이동 구현
2) 기사님 어플 없이 키마레 주소 자동 탐지
3) 택배를 놓는 방법 (관성, 비스틈한 바구니, 레버바구니)
4) 로봇 모용 (트레일러 모양, 타워 모양)

* 종건, 현규 담당

* 건형, 성윤, 재성 담당
* 자연스러운 이동 / 지도 일부 조정 가능
