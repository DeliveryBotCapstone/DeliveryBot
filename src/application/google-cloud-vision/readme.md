애플리케이션의 전체 화면 초기 구상으로, 각 기능당 화면을 나누어 디자인하였음<br />
도움말과 같은 텍스트는 한림대 의료원에서 제공한 무료 글꼴인 한림 고딕체를 이용함<br /><br/>
각 화면들 순서대로
* 애플리케이션 실행시 처음으로 나오는 스플래시 화면
* 송장 촬영 혹은 송장 이미지 업로드를 위한 화면
* 송장 인식여부와 배달 과정에서 택배 수취인에게 SMS 발송 화면
* SMS 발송과 수화물 배달 완료화면
